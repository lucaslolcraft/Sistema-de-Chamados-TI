# 🎫 Sistema de Chamados TI

Sistema web para gerenciamento de chamados de TI, desenvolvido com backend em **Spring Boot** e frontend em **Vite + React**, com foco em aprendizado e estruturação de aplicações web modernas.

---

## 📋 Sobre o Projeto

O **Sistema de Chamados TI** é um projeto criado com o objetivo de estudar e aplicar conceitos de desenvolvimento backend e frontend, como APIs REST, organização de código, comunicação cliente-servidor e estruturação de projetos.

O sistema foi iniciado para auxiliar na organização de demandas de TI, porém **não se encontra finalizado nem em uso produtivo**, servindo como projeto educacional e de portfólio.

---

## 🏗️ Estrutura do Projeto

### 🔧 Backend
- API REST desenvolvida em **Spring Boot**
- Projeto criado utilizando a IDE **Eclipse**
- Estruturação de endpoints para gerenciamento de chamados
- Separação por camadas (controllers, services, repositories)
- Preparação para integração com banco de dados

### 🎨 Frontend
- Desenvolvido com **Vite**
- Interface básica em **React**
- Utilizado principalmente para testes da API
- Comunicação com o backend via requisições HTTP

## 🚀 Como Começar

### 📌 Pré-requisitos
- Java JDK 17 ou superior
- Maven
- Node.js (versão 18 ou superior)
- Git
- MySQL

---

### 📥 Instalação

Clone o repositório:

```bash
git clone https://github.com/lucaslolcraft/Sistema-de-Chamados-TI.git
cd Sistema-de-Chamados-TI
```

## ▶️ Executando o Backend
- Abra a pasta do backend no Eclipse
- Aguarde o Maven baixar as dependências
- Execute a aplicação Spring Boot
- A API estará disponível em:
```bash
http://localhost:8080
```

## ▶️ Executando o Frontend
1. Acesse a pasta do frontend:
```bash
cd frontend
```
2. Instale as dependências:
```bash
npm install
```
3. Execute o projeto:
```bash
npm run dev
```
4. A aplicação estará disponível no endereço exibido no terminal (geralmente http://localhost:5173)
